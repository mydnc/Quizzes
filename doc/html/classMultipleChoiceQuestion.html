Detailed Description

The class is used to store and shows multiple choice question which is child class of the Question class.

Fields: m_Choices is a vector of string which keeps the choices of a multiple choce question. m_NumberOfChoices is an integer field which stores how many choices a multiple question has.

SetChoices() an implementation of a full virtual method from the base class Question. It performs push_back on the vector m_Choices to store choices of a multiple choice question in the container.
