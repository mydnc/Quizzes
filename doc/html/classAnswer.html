Detailed Description

The class is used to store three different types of answers of related questions from the user.

Fields: m_AnswerId holds the id of the answer in integer format. m_TextAnswer stores the text answer in string format. m_BoolAnswer stores the answer in format of boolean. m_CharAnswer stores the answer in choices which user picks between printed choices.

GetAnswer() is a pure vurtial function which is implemented in the child classes and returns answer.
